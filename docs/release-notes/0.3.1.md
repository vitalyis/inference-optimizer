# Release Notes: v0.3.1

## Summary

ClawHub / OpenClaw registry metadata: `SKILL.md` frontmatter declares install commands (`cliHelp`), on-disk state paths (`config.stateDirs` and `config.example`), repository links, and `requires.bins` under `metadata.openclaw` so automated security review can align declared footprint with the skill’s behavior.

The hero image in `SKILL.md` uses an absolute `raw.githubusercontent.com` URL so previews work when the pack is rendered without relative asset resolution. The `social-preview.png` asset is 683×381 (linear dimensions two-thirds of the original 1024×571) for a lighter listing preview.
